# B Performance Dashboard

## 💼 Project Summary

An advanced **Power BI dashboard** built to analyze and visualize
merchandise sales performance across regions, customer segments, and
product categories.

This project demonstrates strong skills in **data analytics, business
intelligence, and storytelling with data**.

------------------------------------------------------------------------

## 🔥 Key Highlights

-   Built an interactive **2-page Power BI dashboard**
-   Transformed raw data into **actionable business insights**
-   Designed a modern, executive-ready UI
-   Applied **DAX calculations** for KPIs and metrics

------------------------------------------------------------------------

## 📊 Business Problem

Businesses often struggle to: - Track performance across multiple
regions - Understand customer buying patterns - Identify top-performing
products - Monitor shipping efficiency

This dashboard solves these challenges by centralizing insights into a
single interactive report.

## 📈 Dashboard Insights

### Sales Performance

-   Total Sales: **\$66K+**
-   Strong monthly fluctuations indicating seasonal trends
-   Balanced performance between local and international markets

### Customer Insights

-   Highest engagement from **ages 18--35**
-   Gender-based purchasing trends identified
-   Ratings reveal customer satisfaction gaps

### Product Performance

-   Clothing dominates revenue contribution
-   Clear identification of top-performing product IDs
-   Opportunity to improve low-performing categories

### Operational Efficiency

-   Shipping trends vary across categories
-   Average shipping performance tracked over time
